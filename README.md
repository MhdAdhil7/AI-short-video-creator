# AI Short Video Creator

An AI-powered content generation tool built using Python, Streamlit, and Ollama (Llama 3).

## 🚀 Features

* Generate video titles
* Generate scripts
* Generate descriptions
* Generate hashtags
* Local AI processing using Ollama
* Simple Streamlit web interface

## 🧠 How It Works

1. User enters a topic in the UI
2. Streamlit sends input to Python backend
3. Python sends prompt to Ollama (Llama 3 model)
4. AI generates content
5. Result is displayed in browser

## 🛠️ Technologies Used

* Python
* Streamlit
* Ollama
* Llama 3
* Git & GitHub

## ▶️ How to Run

```bash
streamlit run app.py
```

## 📌 Project Goal

To help content creators generate scripts, titles, descriptions, and hashtags using AI easily without writing complex prompts.
